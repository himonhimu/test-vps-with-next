## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

```bash
# to add new user on Linux
	adduser himon
# add user to sudo group
	usermod -aG sudo himon
# to change user
	sudo su himon
# to run the runner all the time
# install the svc.sh
	sudo ./svc.sh install
# to check the status of svc
	sudo ./svc.sh status
# to start the runner
	sudo ./svc.sh start
# to stop the runner
	sudo ./svc.sh stop
```
